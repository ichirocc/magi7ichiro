## 集計（合格率・静的）
- **ネイティブ版**：◎/○ が大多数。△は主に**実行時要確認**（読み上げ全面・文字200%・回転/分割・インセット・動きの抑制）。× はなし。
- **Web版**：機能網羅・モバイル対応は良好だが、**専門用語露出**・**開発指標露出**・**装飾(三賢人/TapGame)**・**スキルグループ非対応**で操作面の減点。
- **結論**：全画面・オブジェクト・ポップアップ・メッセージで **ネイティブ版が同点以上**。残課題は機能差ではなく**実機での非機能検証（△）**と、コードで潰せる **A6 動きの抑制**／**読み上げ全面**。

> 注：本表は静的検証。実行時項目（崩れ/回転/kill/性能/読み上げ）は実機（`v6-debug-apk`）で `ux_test_checklist.md` の手順により確定する。
